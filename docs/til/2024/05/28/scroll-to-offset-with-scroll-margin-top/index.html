Scroll to offset with scroll-margin-top

May 28, 2024

TIL there are some CSS properties that you can set to adjust where a browser scrolls to when scrolling to a div without ruining the perfect spacing of your page!

Just add:

.my-element {
  scroll-margin-top: 60px;
}

And then this will scroll to 60px above the element:

document.querySelector(".my-element")
  .scrollIntoView({ block: "start" })
